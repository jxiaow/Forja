真实工作流命令 Real workflow commands

Qt 与 SDK 项目的常用流程，来自 Forja CLI 文档。 Common Qt and SDK workflows from the Forja CLI documentation.

Qt Project 
# check environment
$ forja qt status

# initialize project config
$ forja qt init

# switch build mode
$ forja qt use --mode release

# build and run
$ forja qt build
$ forja qt run
SDK Project 
# check SDK configuration
$ forja sdk status

# initialize SDK project
$ forja sdk init

# build SDK project
$ forja sdk build

# inspect available projects
$ forja qt projects
