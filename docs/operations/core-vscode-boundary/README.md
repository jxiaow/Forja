# Operations Template

This directory is the operations document template for long-running / multi-stage / continuous tasks.

Generate command:

```bash
node harness/core/operations/create-operation-docs.js core-vscode-boundary
```

Defaults to generating in `docs/operations/core-vscode-boundary/`.

Generated files:

- `current-core-vscode-boundary.md`
- `core-vscode-boundary-board.md`
- `core-vscode-boundary-matrix.md`
- `core-vscode-boundary-decisions.md`
