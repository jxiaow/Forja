# Web Vue Node Profile

这是 Web/Vue/Node 类项目的可选 profile。它只提供生成方向，不是默认通用模板。

可选规则：

- `backend-route-patterns.md`：后端路由、认证、错误处理、数据访问、挂载入口
- `theme-patterns.md`：主题变量、设计 token、响应式样式
- `vue-component-patterns.md`：组件组织、状态生命周期、composable、router/store 接入
- `vue-event-patterns.md`：emit、watch、computed、异步状态

采用前先确认仓库确实存在：

- Vue 或类似组件框架
- Node 或类似后端路由层
- 主题变量或设计 token
- router/store/composable 等稳定模式

如果项目是 CLI、库、后端服务、移动端、嵌入式或基础设施仓库，不要使用本 profile。
